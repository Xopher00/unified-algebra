# UniAlg Haskell Branch

This is the entry point for the Haskell reference guide.

- [Overview](haskell/overview.md) explains the Haskell pipeline.
- [Core model](haskell/core-model.md) covers `TArr`, `Shape`, functor mapping,
  arrow combinators, and optics.
- [Backends](haskell/backends.md) documents backend JSON specs and custom
  backend mappings.
- [Generating code](haskell/generating-code.md) shows how to compose backend
  primitives and emit code without recursion schemes.
- [Writing recursive architectures](haskell/writing-architectures.md) is the
  guide for adding recursion-scheme modules under `explore/archs/`.
- [Reference](haskell/reference.md) keeps the module map and known open issues.
